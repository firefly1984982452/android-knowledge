====================文件结构信息====================
open_sdk_***.jar文件：SDK jar文件，使用时必须引入工程。(位于目录jar下）
jar：SDK的jar包目录
sample：示例代码源码目录。
doc：SDK使用说明文件目录。
tools:获取APK签名工具目录。

====================示例代码文件====================
sample: opensdk的示例工程源码文件。
open_sample.apk：示例代码安装文件（位于目录sample/bin下）。
MainActivity.java：	Andriod_SDK_V1.7使用的示例代码（位于目录sample/src/com/tencent/sample下）
